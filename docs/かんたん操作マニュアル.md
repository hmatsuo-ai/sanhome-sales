# サンホーム営業管理システム かんたん操作マニュアル

このマニュアルは、ITに慣れていない方でも使えるように、操作を順番に説明しています。

## 1. 最初にやること（ログイン）

1. ブラウザでシステムを開く
2. メールアドレスを入力
3. パスワードを入力
4. 「ログインする」を押す

ログインできないときは、入力ミス（全角/半角、スペース）を確認してください。

## 2. 経費を登録する

1. 左メニューの「経費精算」を押す
2. 画面右上の「新規登録」を押す
3. 次を入力する
   - 日付
   - カテゴリ
   - 金額
   - メモ（必要なとき）
   - 領収書画像（必要なとき）
4. 「保存」を押す

### 経費を期間で見る

1. 経費精算画面の上部で、開始日と終了日を選ぶ
2. その期間のデータだけ表示される
3. 「今月」ボタンで今月表示に戻せる

### 経費を絞り込む（表示するものだけにする）

1. 一覧の「担当者」または「カテゴリ」を押す
2. 表示したい項目にチェックを付ける
3. 不要なときは「すべて」を押して解除する

## 3. 担当者ごとの経費合計を見る

1. 経費精算画面の「個人別合計一覧」ボタンを押す
2. 開始日と終了日を選ぶ
3. 担当者ごとの合計金額と件数を確認する

必要に応じて、列名（担当者 / 合計金額）を押して並べ替えできます。

## 4. 売上を登録する

1. 左メニューの「売上管理」を押す
2. 右上の「新規登録」を押す
3. 次を入力する
   - 契約日
   - 決済日
   - 担当者（1名以上）
   - 担当者ごとの配分比率（合計100%）
   - カテゴリ
   - 案件名
   - 売上金額
   - 粗利金額
4. 「保存」を押す

## 5. 売上を確認する

### 期間を変える

1. 売上管理画面上部の開始日・終了日を変更
2. 一覧とサマリーが自動で更新される

### タブを切り替える

- 「事業利益・仲介」: メインの売上
- 「賃貸（いい部屋）」: 賃貸の売上

### サマリーを切り替える

1. サマリー上部の「全体 / グループ / 個人」を押す
2. 必要に応じて、グループや個人を選ぶ
3. 「選択期間」と「半期通算」の数字を確認する

### 一覧を絞り込む

一覧のヘッダーから次の条件で絞り込みできます。
- 担当者
- 案件名
- カテゴリ
- 決済状況

## 6. スケジュールを使う

1. 左メニューの「スケジュール」を押す
2. 「追加」ボタンで予定を登録
3. 予定を押すと編集できる
4. 不要な予定は削除できる

表示は「全員 / グループ / 個人」で切り替えできます。

## 7. 管理者向け（設定画面）

管理者ユーザーだけ「設定」が表示されます。

- ユーザーの追加・編集
- グループの追加・削除
- ユーザーの所属グループ設定

## 8. 困ったとき

- 画面が更新されない: ブラウザの再読み込み（F5）
- 保存できない: 必須項目の入力漏れがないか確認
- ログインできない: メール/パスワードを再確認
- それでも解決しない: 管理者へ連絡
