# サンホーム 営業統合管理システム — 使い方の説明書

この資料は、**パソコンやブラウザにあまり慣れていない方**でも、日々の操作に困らないようにまとめた説明書です。  
専門用語はできるだけ使わず、**画面上の言葉と操作の順番**を中心に書いています。

---

## 1. このシステムでできること

営業の日常業務を、**インターネット上の1か所**でまとめて扱えます。

| やりたいこと | 使う画面（左のメニュー） |
|-------------|-------------------------|
| 今月の売上や経費のざっくりした数字を見る | **ダッシュボード** |
| 交通費などの経費を登録する | **経費精算** |
| 案件の売上・粗利を登録する | **売上管理** |
| 訪問や打ち合わせの予定を入れる | **スケジュール** |
| メールアドレスやパスワードを変える（管理者はメンバー管理も） | **設定** |

---

## 2. はじめる前に（最低限おさえておくこと）

### 2.1 必要なもの

- **インターネットに繋がるパソコンまたはスマートフォン**
- **画面を開くためのソフト（ブラウザ）**  
  例：Google Chrome、Microsoft Edge、Safari など、普段お使いのもので結構です。
- **会社から案内されたアドレス（URL）**  
  例：`https://（会社が指定した住所）` のような形です。メールや紙で共有されます。

### 2.2 画面上のいくつかの言葉

- **クリックする**… パソコンでマウスの左ボタンを1回押すことです。（タップする、と同じ意味です）
- **入力する**… 文字をキーボードで打つことです。
- **ボタン**… 色が付いていて、押すと何かが始まる場所です。「保存」「ログインする」などがあります。
- **メニュー（一覧）**… 画面の左に並んでいる「ダッシュボード」「経費精算」などの一覧です。**行きたい画面をここから選びます。**

---

## 3. ログインのしかた

### 3.1 手順

1. ブラウザを開き、アドレス欄に **会社から教えてもらったURL** を入力して Enter キーを押す（またはリンクを開く）。
2. **ログイン画面**が表示されたら、次の2つを入力します。
   - **メールアドレス**（会社で使うメール）
   - **パスワード**（6文字以上。＝マークの付いた欄は、打った文字が隠れて見えますが正常です）
3. **「ログインする」** のボタンを押します。
4. うまくいけば **ダッシュボード**（数字が並んだ最初の画面）が開きます。

メールやパスワードが違うと、**赤い文字やメッセージ**が出ることがあります。そのときは、大文字・小文字・全角半角をもう一度確認してください。

### 3.2 初めて自分でアカウントを作る場合

ログイン画面に **「ログイン」と「アカウント作成」** の切り替えがあるときは、**「アカウント作成」** を選び、名前・メール・パスワードを入れて登録できます。

- 登録できるのは **一般の営業担当としての権限** です。**管理者権限が必要な方は、必ず会社の管理者に相談**してください。
- すでに同じメールアドレスが使われていると、「登録できない」と出ることがあります。

### 3.3 ログアウト（終了するとき）

1. 画面の左の下のほうにある **「ログアウト」** を押します。
2. ログイン画面に戻ります。**共用のパソコンでは、作業の最後に必ずログアウト**する习惯をつけると安全です。

---

## 4. 画面の見方

### 4.1 パソコンで見る場合

- **左の青い部分**：メニュー。名前と「営業」か「管理者」が表示されていれば、ログインできています。
- **右の白い広い部分**：今選んでいる画面の内容です。

### 4.2 スマートフォンで見る場合

- 画面上部の **三本線（≡）** を押すと、メニューが開きます。
- 画面の下にも、よく使うメニューのショートカットが並ぶことがあります。

---

## 5. 各画面の使い方（よく使う操作）

### 5.1 ダッシュボード

**何のため？** … 選んだ月や年の **売上・粗利・経費の合計** と、**今日の予定** をざっと見ます。

1. **「月別」または「年別」** を選ぶ。
2. **年月（または年）** を選ぶ。
3. **担当者** で「全員」か、特定の人を選ぶと、その範囲の数字が変わります。

画面上の「今月の〜」と書いてあっても、**いま選んでいる期間の合計** として読んでも大丈夫です。

---

### 5.2 経費精算

**何のため？** … 飲食費・交通費など **経費を登録し、一覧で見る** ためです。

**新しく1件登録する**

1. メニューから **「経費精算」** を開く。
2. 対象の **月** を上で選ぶ。
3. **「新規登録」** を押す。
4. 画面に出た入力欄に入れる。
   - **日付** … お金を使った日。
   - **カテゴリ** … 飲食費・交通費などから選ぶ。
   - **金額** … 税込みの数字（半角がおすすめです）。
   - **メモ** … 店名や用途（なくても登録はできます）。
   - **領収書の写真** … あれば写真を選んで添付できます（**自分で金額などを入力するのが基本**です）。
5. **「保存」** を押して終了。

**削除**

- 一覧の **「削除」** は、**自分が登録した経費** にだけ出ます。押すと消えます。**本当に消してよいか確認の窓** が出たら、内容を読んでから進めてください。

---

### 5.3 売上管理

**何のため？** … **案件ごとの売上・粗利** を登録し、**いつ決済したか** を管理するためです。

1. **「事業利益・仲介」** と **「賃貸（いい部屋）」** のタブで、見たい種類を切り替えます。
2. **期間**（始まりの日〜終わりの日）を指定すると、その期間の一覧が変わります。
3. **「新規登録」** で案件を追加します。
   - **決済日** は必須です。
   - **担当者** は1人以上選びます。**配分比率（％）** を入れなくても、自動で分け方が決まります。
   - **売上金額** と **粗利金額** を入力します。
4. 一覧では **「未決済」「決済済」** を切り替えて状態を管理できます。
5. **ゴミ箱** のマークは削除です。**確認を出してから** 進みます。

---

### 5.4 スケジュール

**何のため？** … **訪問や会議などの予定** をカレンダーに載せます。

1. **全員／グループ／個人** の見方を選びます。
2. **「＜」「今日」「＞」** で週や日を移動します。
3. **「追加」** または **空いているマス** を押して予定を入れます。
   - **タイトル** は必須です。
   - **日付・開始・終了** を入れます。
   - **場所** はなくても構いません。
4. 予定を押すと **内容を直せます**。
5. **自分が登録した予定には「✕」** が出ます。押すと削除できます（確認あり）。

---

### 5.5 設定

**営業の方**

- **画面の見た目（明るい／暗い）** の切り替えと、**メールアドレス・パスワードの変更** ができます。
- より細かい個人の実績や経費の一覧は、売上・経費の画面から **「詳細を表示」** などで開ける **個人ページ** でも確認できます。

**管理者の方**

- **新しいユーザーの登録**（名前・メール・初期パスワード・権限）。
- **グループの作成・削除**、**誰がどのグループか** の設定。
- **データベース接続** など、運用向けの情報（通常の日々の業務ではなく、障害確認などで使うことが多い内容です）。

---

## 6. 左メニューの下部にある項目

- **API連携確認** … サーバーとの通信が順調かを **動作確認** する画面です。日常業務では使う必要は少ないですが、「急に数字が出ない」ときに担当者が確認することがあります。
- **修正依頼** … 画面の使いにくさや不具合を **文章で送る** ための場所です。何が起きたかを具体的に書くと伝わりやすいです。

---

## 7. うまくいかないとき（簡単なチェック）

| 気になること | まず試すこと |
|-------------|-------------|
| ログインできない | メールの綴り・パスワードの大文字小文字を確認。パスワードを忘れた場合は **管理者にリセットを依頼** してください。 |
| 真っ白な画面・ずっと読み込み | 一度ブラウザを閉じてから、もう一度URLを開く。それでもダメなら時間をおいて再試行。 |
| 保存しても反映されない | **「保存」ボタンを押したか**、**必須の欄（日付や金額など）に抜けがないか** を確認。 |
| 削除してしまった | このシステムには **「取り消し」で元に戻す** 機能は基本的にありません。管理者に相談してください。 |

---

## 8. 大切なお願い

- **パスワードは他人に教えない**でください。推測されやすい誕生日だけのパスワードは避けましょう。
- **間違った数字を登録した**と気づいたら、できるだけ早く直すか、上司・管理者に相談してください。
- **共用PC** では、席を離れる前に **ログアウト** してください。
