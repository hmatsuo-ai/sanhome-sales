# サンホーム営業管理システム やさしい仕様書

この資料は、ITに詳しくない方向けに「このシステムで何ができるか」をやさしく説明したものです。

## 1. このシステムでできること

- 日々の仕事に必要な **売上・経費・予定** を1つの画面で管理できます。
- 担当者ごと、グループごと、全体で数字を確認できます。
- 管理者は、ユーザーやグループの管理もできます。

## 2. 使える画面

### ログイン
- メールアドレスとパスワードでログインします。
- ログアウトは左メニューの「ログアウト」です。

### ダッシュボード
- 売上・粗利・経費の合計を確認できます。
- 当日の予定も確認できます。

### 経費精算
- 経費を登録できます（領収書画像の添付も可能）。
- 期間（開始日〜終了日）を選ぶと、その期間の経費だけ表示できます。
- 担当者・カテゴリで「表示するものだけ」を絞り込めます。
- 日付は昇順/降順の切替ができます。
- 金額・メモでも並べ替えできます。

### 経費 個人別合計
- 担当者ごとの経費合計を一覧で見られます。
- 期間（開始日〜終了日）を選んで、集計対象を変更できます。
- 担当者名・合計金額の並べ替えができます。

### 売上管理
- 売上を登録できます（担当者を複数選択可能）。
- 期間（開始日〜終了日）を選んで、表示対象を変えられます。
- 「事業利益・仲介」「賃貸（いい部屋）」のタブ切替ができます。
- サマリーは **全体 / グループ / 個人** を切り替えて見られます。
- 一覧は、次の条件で絞り込みできます。  
  - 担当者
  - 案件名
  - カテゴリ
  - 決済状況（すべて / 決済済み / 未決済）

### スケジュール
- 予定の登録・編集・削除ができます。
- 全員 / グループ / 個人で見方を切り替えられます。

### 設定（管理者のみ）
- ユーザーの追加・編集・削除
- グループの追加・削除
- ユーザーの所属グループ設定

## 3. 権限について

- **営業担当**: 日常業務の画面（ダッシュボード、経費、売上、スケジュール）を利用
- **管理者**: 上記に加えて「設定」画面を利用

## 4. データの基本ルール

- 売上は担当者を複数設定できます。粗利は担当者ごとに配分されます。
- 経費・予定は登録した人に紐づきます。
- 削除前には確認メッセージが表示されます。

## 5. よくある操作の考え方

- 「並べ替え」: 表示の順番だけを変える
- 「絞り込み（フィルタ）」: 条件に合うものだけを表示する
- 「サマリー」: 合計値を見やすくまとめた表示
